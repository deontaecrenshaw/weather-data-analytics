# Weather Data Analytics Platform

## Overview

This project demonstrates an end-to-end weather analytics pipeline developed using Python, SQL, SQLite, and the NOAA API.

Weather observations are collected, stored in a SQLite database, queried using SQL, cleaned for analysis, and visualized using Python and Excel.

---

## Technologies Used

- Python
- SQL
- SQLite
- NOAA API
- Microsoft Excel

---

## Skills Demonstrated

- API Integration
- Database Design
- SQL Queries
- Data Cleaning
- Data Visualization
- Python Programming
- Data Analysis

---

## Project Features

- Download weather observations from NOAA
- Store weather data in SQLite
- Query data using SQL
- Export data for analysis
- Generate charts and visualizations
- Analyze temperature and humidity trends
